== Purpose

The `planner` plugin encapsulates a Planning Expert that decomposes a natural language
intention into a structured execution plan (EPICs â†’ User Stories â†’ Tasks) via LLM.

== Invocation

  ./gradlew decomposeIntention -Pintention="description" [-PspecsDir=specs/]

Output: stdout in structured format `[PLAN] [EPIC] [US] [TASK]`.

== Architecture

- `DecomposeIntentionTask` : Gradle task entry point
- `IntentionPlanner` : builds prompt, calls OllamaBridge, retries 3x, parses JSON â†’ Plan
- `SpecReader` : reads .adoc specs from directory, injects into prompt (token budget ~2000)
- `OllamaBridge` : LangChain4j chat model wrapper for local Ollama/DeepSeek-v4-pro
- `StdoutFormatter` : formats Plan for human-readable stdout
- `Plan` / `PlanningContext` : domain data classes
