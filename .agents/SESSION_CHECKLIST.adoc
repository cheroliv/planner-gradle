= SESSION_CHECKLIST — planner-gradle

== Hook d'Ouverture — Avant de commencer une session

. [ ] Lire `PROMPT_REPRISE.adoc` — contexte de la session precedente
. [ ] Lire `AGENT.adoc` — regles absolues (4 regles)
. [ ] Lire `.agents/INDEX.adoc` — roadmap, etat des EPICs, portefeuille
. [ ] Verifier l'etat git (`git status`) — branche, fichiers modifies
. [ ] Verifier que le build compile (`./gradlew :planner-plugin:compileKotlin`)
. [ ] Si le plugin a ete modifie : `./gradlew :planner-plugin:publishToMavenLocal`

== Pendant la session

. [ ] Respecter la Regle 1 — AUCUN commit sans permission explicite
. [ ] Respecter la Regle 2 — publishToMavenLocal OBLIGATOIRE apres modif code source
. [ ] Respecter la Regle 3 — ZERO secret dans build.gradle.kts
. [ ] Verifier que chaque modification compile avant de continuer

== Hook de Fermeture — Procedure 6 etapes (voir `SESSION_PROCEDURE.adoc`)

. [ ] 1. Rapport de session → `.agents/sessions/NNN-sujet.adoc`
. [ ] 2. Mise a jour `PROMPT_REPRISE.adoc` — contexte reprise N+1
. [ ] 3. Mise a jour `INDEX.adoc` — numero de session, roadmap, statut EPICs
. [ ] 4. Mise a jour `SESSIONS_HISTORY.adoc` — ajouter entree
. [ ] 5. Archivage `COMPLETED_TASKS_ARCHIVE.adoc` — taches terminees
. [ ] 6. Verification — `git status`, absence de secrets, build compile

== Commandes de Verification

[source,shell]
----
# Verifier l'etat git
git status

# Verifier l'absence de secrets
grep -r "TOKEN\|SECRET\|PASSWORD\|API_KEY" planner-plugin/ --include="*.kts" --include="*.properties" --include="*.yml"

# Verifier que le build compile
./gradlew :planner-plugin:compileKotlin

# Publier en local si modif
./gradlew :planner-plugin:publishToMavenLocal
----
